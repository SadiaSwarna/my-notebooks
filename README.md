# Retail Sales Dashboard (Power BI Project)

## Overview

This project is a Retail Sales Dashboard built using Power BI. It analyzes sales performance across regions, products, customers, and sales teams.

## Key Features

* Total Sales, Profit, Profit Margin KPIs
* Monthly Sales Trend
* Top 10 Products Analysis
* Customer Segmentation
* Salesperson Performance

## Tools Used

* Power BI
* DAX
* Excel

## Dashboard Pages

1. Executive Summary
2. Product Insights
3. Customer Insights
4. Sales Team Performance

## Insights

* Identified top-performing products and regions
* Analyzed customer segments
* Evaluated sales team performance

* Power BI file (.pbix)
